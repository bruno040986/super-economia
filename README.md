# Supermercado Economia - Site Institucional

Site institucional oficial do **Supermercado Economia**, localizado no bairro Parque Esplanada II, em Valparaíso de Goiás. Desenvolvido com foco em alta performance, responsividade (mobile-first) e facilidade de manutenção.

## 🚀 Tecnologias Utilizadas

- **HTML5 & CSS3:** Semântica e estilos modernos, utilizando variáveis CSS e Flexbox/Grid para layouts responsivos.
- **JavaScript (Vanilla):** Lógica para menu interativo, animações de scroll e componentes dinâmicos.
- **Vite:** Ferramenta de build super rápida para desenvolvimento e otimização de assets.
- **FontAwesome:** Ícones vetoriais.
- **Google Fonts:** Tipografia moderna (Anton e Montserrat).

## 🎯 Funcionalidades

- **Design Mobile-First:** Experiência otimizada para smartphones, tablets e desktops.
- **Performance:** Imagens no formato moderno `.webp` e carregamento otimizado (`loading="lazy"`).
- **SEO Ready:** Meta tags essenciais preparadas, estruturação semântica e Open Graph integrado.
- **Menu Fixo e Barra Flutuante:** Acesso rápido aos contatos e redes sociais durante a navegação.
- **Integração com Google Maps:** Facilidade para o cliente traçar rotas direto para a loja.

## 📦 Como rodar o projeto localmente

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone o repositório:
   ```bash
   git clone https://github.com/SeuUsuario/supermercado-economia.git
   ```
3. Acesse a pasta do projeto:
   ```bash
   cd supermercado-economia
   ```
4. Instale as dependências:
   ```bash
   npm install
   ```
5. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
